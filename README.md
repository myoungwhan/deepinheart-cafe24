# DeepInHeart

Flutter application for counseling and fortune-telling services — client app and counselor dashboard.

## Documentation

- **[Setup & Run Guide](docs/SETUP_GUIDE.md)** — Prerequisites, install, configuration, run, and build.
- **[Project Structure](docs/PROJECT_STRUCTURE.md)** — Architecture, folder layout, and conventions.

## Quick start

```bash
flutter pub get
flutter run
```

See [docs/SETUP_GUIDE.md](docs/SETUP_GUIDE.md) for full setup and configuration (API, Firebase, Agora, etc.).

## Tech stack

- **Flutter** (Dart 3.7+)
- **State:** Provider, GetX
- **Backend:** REST API
- **Calls:** Agora (voice/video)
- **Auth & messaging:** Firebase, Kakao, Google, Apple (as configured)
